# ⚠️ ARCHIVE NOTICE: PROJECT MIGRATION TO TSN

### Tinh Linh Programming Language is now deprecated and succeeded by TSN (TSN Standard Notation).

The **Sao Tin Developers** team has officially decided to cease development of **Tinh Linh Lang** to focus all resources on **TSN (TSN Standard Notation)**. 

**TSN** is the direct successor to Tinh Linh, designed to provide a more robust ecosystem, a familiar TypeScript-like syntax, and professional-grade memory management.

---

## 🚀 Why the shift to TSN?

While Tinh Linh achieved significant milestones—including full **Self-hosting** capabilities—we realized that developers need a language that bridges the gap between high-level syntax and low-level performance more seamlessly.

| Feature | Tinh Linh Lang | TSN (Successor) |
| :--- | :--- | :--- |
| **Syntax** | Lithium (.li) | **TypeScript Standard** |
| **Memory Management** | Basic / Manual | **Ownership & Move Semantics** |
| **Performance** | High (LLVM) | **Ultra-High (Zero-overhead)** |
| **Standard Library** | Experimental | **Production-ready (OS, FS, Hash, Memory)** |
| **Type System** | Static | **Advanced Generics & Recursive Inference** |
